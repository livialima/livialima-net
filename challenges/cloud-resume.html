The Cloud Resume Challenge

The cloud resume challenge is a hands-on challenge created by Forrest Brazeal. The idea is to create your own resume using essential cloud skills.

My Background: I found this challenge when I was preparing to publish my website and the simplicity was what hooked me. I already have IT experience, which disqualifies me for a referral from Forrest. But since my initial goal was to simply put my website out there, there's no harm in trying.

My Progress: The challenge consists of 16 proposed steps. I did the first 6 steps to start blogging and, for that, it's good enough. But I wanted to finish the thing, so here we are.
