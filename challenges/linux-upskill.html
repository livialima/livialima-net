Linux Upskill Challenge

The Linux Upskill Challenge is a hands-on Linux course created by Steve Brorens. The course is run via Reddit and each weekday a new lesson is posted there; it's an awesome place to discuss and get help.

The month-long course is free and fully open source. You can check its source out on GitHub. The course restarts on the first Monday of next month, so don't worry if you missed some of the lessons.

My Twist: I'm already a Linux user for many years but (to my own shame) I never got to the SysAdmin level. I can do a lot, just not enough for some cases. I'm taking the oportunity to UP my SKILL (see what I did there?), so my notes/posts will probably get a little further in many of the topics covered by the challenge.
