Day 0 : Preparing the environment

This is part of the #LinuxUpSkillChallenge. Don’t forget to also check out the Official Instructions and the Reddit posts.

Objectives of the day

  1. Sign up to a cloud provider: you can use AWS, Google Cloud, Azure, Digital Ocean, Linode, etc. I’m already signup with AWS, so I’m using that.
  2. Launch an Ubuntu instance: Follow the tutorial How to launch an EC2 instance, select the latest version of Ubuntu and set the security group to All Traffic / Anywhere.
  3. Connect to the instance using SSH: Follow the tutorial on Putty Basics or (if you’re using OpenSSH) simply copy the command from AWS instructions.

Administrative task: Update/Upgrade the system

Once connected, submit the commands:

sudo apt update

And then:

sudo apt upgrade -y

Done!
