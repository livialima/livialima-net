Day 2 : Basic navigation

This is part of the #LinuxUpSkillChallenge. Don't forget to also check out the Official Instructions and the Reddit posts.

Objectives of the day

  1. Navigate the file system with: pwd cd cd / cd .. cd - cd ~ ls ls -l ls -a
  2. Deal with files and directories try the commands: mkdir rmdir touch rm mv cp
  3. RTFM commands: man apropos tldr
  4. Administrative task: Customize your prompt export PS1="\u@\h \W\$" For more about prompt customization, check out the Bashrc Generator and Customize & Colorize Your Bash Prompt/Terminal.

Done!

In addition to the extensions of the challenge, I also recommend the Bash Reference Manual.
